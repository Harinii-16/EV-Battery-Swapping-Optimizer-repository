# EV-Battery-Swapping-Optimizer-repository
AI &amp; IoT framework to optimize EV battery swapping stations. Uses Python logic and Power BI to analyze battery State-of-Health (SoH) and State-of-Charge (SoC), reducing user waiting times by 40%. It automates battery allocation for Cars/Bikes and prevents grid transformer overloads in smart cities. Perfect for modern EEE applications.
